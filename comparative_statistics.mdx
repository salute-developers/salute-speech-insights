# Сравнительная статистика

При сборе статистики анализируются только те фрагменты звонка — реплики, — в которых есть распознанная речь. Статистика по эмоциям собирается только по тем репликам, которые длиннее двух секунд или в которых больше двух слов.

Показатель `emo_score` – это разница между вероятностью позитива и негатива. Принимает значения от -1 до 1, где -1 — это уверенный негатив, а 1 — это уверенный позитив.

Во всех показателях, где есть `emo_score_weighted_by` или `emotion_weighted_by` (отмечены *), средний `emo_score` или средняя эмоция считаются с учетом веса данной реплики относительно всего канала. Вес может считаться по количеству слов или по длительности речи. Например, если в звонке 2 реплики: в первой 9 слов и эмоция — нейтраль (0), а во второй 1 слово и эмоция — позитив (1), то вес первой — 0,9, а второй — 0,1. Средняя эмоция, взвешенная по количеству слов, считается так: `0 x 0,9 + 1 x 0,1 = 0,1`.

## Статистики перебиваний

Перебивание — часть диалога, в которой клиент и оператор активно перебивают друг друга, то есть есть несколько фрагментов одновременной речи. Одиночный короткий фрагмент одновременной речи не считается перебиванием

|**Идентификатор показателя**|**Описание показателя**|**Единица измерения**|
|----|-----|-----|
| `dialog_interruptions_count`| Количество перебиваний в диалоге| Штуки   |
| `dialog_interruptions_speech_length_sum`| Суммарная длительность одновременной речи внутри всех перебиваний| Секунды |
| `dialog_interruptions_speech_length_percentage`| Доля одновременной речи при перебиваниях в звонке целиком (`dialog_interruptions_speech_length_sum` / `call_duration`) |Проценты — от 0 до 1|
| `dialog_interruptions_raw_speech_length_sum`| Суммарная длительность одновременной речи| Секунды |
| `dialog_interruptions_raw_speech_length_percentage`|Доля одновременной речи в звонке целиком (`dialog_interruptions_raw_speech_length_sum` / `call_duration`)|Проценты — от 0 до 1|
| `dialog_interruptions_in_anybody_speech_percentage`| Доля одновременной речи при перебивании во время того, когда кто-нибудь говорит (`dialog_interruptions_speech_length_sum` / `dialog_anybody_speech_length_sum`)| Проценты — от 0 до 1|
| `dialog_interruptions_in_customer_speech_percentage`| Доля одновременной речи при перебивании в речи клиента (`dialog_interruptions_speech_length_sum` / `customer_speech_length_sum`)| Проценты — от 0 до 1|
| `dialog_interruptions_in_agent_speech_percentage`| Доля одновременной речи при перебивании в речи оператора (`dialog_interruptions_speech_length_sum` / `agent_speech_length_sum`)| Проценты — от 0 до 1|
| `dialog_interruptions_list`| Составной параметр, включающий описание каждого интервала перебивания| -       |

### Разница в скоростях по двум каналам
